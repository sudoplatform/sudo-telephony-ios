    Possible states for a SMS/MMS message. QUEUED - Outbound message queued by telephony vendor ready for sending to carrier. SENT - Outbound message sent by telephony vendor to carrier. DELIVERED - Outbound message acknowledged as delivered by carrier. UNDELIVERED - Outbound message acknowledged as undelivered by carrier. FAILED - Outbound message acknowledged as failed by carrier.
